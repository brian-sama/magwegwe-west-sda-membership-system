Magwegwe West SDA Membership System - cPanel Deployment Instructions (Simple PHP)

This package contains:
1. Frontend build (upload contents to `public_html/`).
2. `api/` folder (simple PHP backend).

---
STEP 1: DATABASE SETUP
---
1. In cPanel, open `MySQL Databases`.
2. Create a database named `church_membership`.
3. Create a user and grant all privileges to the database.
4. Import `database_schema.sql` (if provided) or create the tables.

---
STEP 2: BACKEND DEPLOYMENT (API)
---
1. Upload the `api/` folder into `public_html/api`.
2. Edit `api/config/database.php` and set your cPanel database credentials:
   $host = "localhost";
   $db   = "church_membership";
   $user = "your_cpanel_user";
   $pass = "your_cpanel_password";

---
STEP 3: FRONTEND DEPLOYMENT
---
1. Upload all files from the frontend build to `public_html/`.
2. The system is pre-configured to look for the API at yourdomain.com/api/.

---
STEP 4: VERIFICATION
---
1. Visit yourdomain.com in the browser.
2. Try logging in with your credentials.
